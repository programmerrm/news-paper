# News Flash 71

A modern news web application built with Next.js and Tailwind CSS, focused on clean UI, fast performance, and a smooth reading experience.

# Description

News Flash 71 is a frontend-focused news platform that displays the latest news in a simple, responsive, and user‑friendly interface. It’s designed to be fast, minimal, and easy to extend with APIs or backend services later.

# Features

- Clean and modern UI
- Fully responsive design
- Fast performance with Next.js
- Styled using Tailwind CSS
- Easy to customize and scale

# Tech Stack

- Language: JavaScript
- Framework: Next.js
- Frontend: HTML, CSS
- Styling: Tailwind CSS

# Installation

- Step 1: Install dependencies npm install

- Step 2: Start the development server npm run dev

- Step 3: Now open your browser and go to http://localhost:3000
